# 实现文章搜索功能

## 后端实现
1. **在 post.ts 中添加搜索接口**
   - 添加 GET /api/post/search 接口
   - 支持通过 query 参数接收搜索关键词
   - 使用 SQL LIKE 语句实现标题模糊搜索
   - 保持与现有接口相同的返回格式

## 前端实现
1. **在 post.html 中添加搜索组件**
   - 在工具栏中添加搜索输入框
   - 添加搜索按钮
   - 添加重置按钮（可选）

2. **修改前端逻辑**
   - 添加 searchData() 函数来处理搜索请求
   - 修改 loadData() 函数，支持传入搜索关键词
   - 实现搜索按钮的点击事件
   - 实现重置按钮的点击事件，清空搜索框并重新加载所有数据

## 技术细节
- 后端使用 SQL LIKE '%关键词%' 实现模糊搜索
- 前端使用 Webix UI 库的组件
- 保持与现有代码风格一致
- 确保搜索功能与现有功能（新增、修改、查看详情）兼容
